about

I am a new graduate from the University of Waterloo, Canada. I have a bachelor degree in computer science with a computational fine art option. I have experiences in mobile, web, and desktop applications development and have a passion in computer graphic and digital fine art.

Some other fun facts about me: I like cooking and baking. I love my dog Coco and my obese cat Momo.

Project

Ray Tracer (Advance)

Implemented using c++, including more advance techniques such as: texture mapping, bump mapping, adaptive anti-aliasing, phong shading, depth of field, motion-blur and more.

Ray Tracer (Basic)

The basic version of a ray tracer, implemented using c++. Including: 3D primitives rendering, reflection, model hierarchye.

Barcraft 3D Sandbox

The first OpenGL project I worked on, was implemented by using C++ and OpenGL.

3D humanoid puppet

Implemented by using c++ and OpenGL, where the body movement can be controlled interactively. The puppet is based on Boo from "The Monster Inc".

Body Input Visualization

Implemented by using Processing, which uses motion as inputs to control the digital graphics output in a creative and artistic way. Side note: the video is pretty silly to watch...

Neural Network Simulation

Implemented by using Processing, a simulation on our brain activity in an abstract and creative way.

The Galaxy Lake

Implemented by using Processing, a visual illusion of colorful fishes swimming in the galaxy. Fishes represent the spirit of life, wondering and glowing in the dark.

ER Diagram Editor

Entity Relationship Diagram Editor implemented by Java. It is based on the model view controller architecture.

Spotify Music Management

A web-based tool for associating ratings, tags and add a new playlist to the Spotify account from the result of a search.

Tetris Game

The simple classic tetris game implemented by using Java.

I Love Take-Out

An IOS app that allows user to order take-out based on their location.

Numericons

An Android game where the player is given a target number, and receives a stream of numbers where they can add, subtract, multiply, and divide them together to reach the target number.

Text Smoking Effect

A quick practice of the particle system, which was implemented by using Processing.

Flight Path

Another quick practice of data visualization that simulate international flight paths. This was implemented by using processing.
